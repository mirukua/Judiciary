通过大模型生成伪监督数据  data_produce
"""将法律文书拆分成 事实 法条 结论"""


1.读取文书文本
按段落拆分

2.结构化拆解
使用规则 / 正则 / NLP 工具提取：
事实段落 → 事实
法条引用 → 法条
判决结果 → 结论

3.构建 input
直接使用原文或抽取后的事实文本
可合并文书摘要（例如事实+证据）

4.构建 instruction
根据任务类型自动生成：
模板：“请从下文提取案件事实。”
模板：“请根据事实判断罪名并说明理由。”
可用 LLM 做多样化改写生成自然指令

5.构建 output
用规则提取文本
或用原判决中法院意见 + 法条组合
可生成多种粒度：纯文本 / 分条 / 三段论结构

6.去重 & 多样化
对 instruction 进行改写生成多样化任务
对 output 可生成不同层次（简略 / 详细 / 推理步骤）
